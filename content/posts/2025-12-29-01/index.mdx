又到了写年终总结的时候了，今年第一篇依旧先是一句话点评我玩了什么，还是按照我的游玩顺序排列。同时今年还会略微点评一下我的各方面年度最佳游戏（范围是我今年玩过的游戏，奖项参考了狗蛋的部分奖项），以及这些开发商做不明白肉鸽就别做了。

# **Fate/Samurai Remnant 9/10**

月丑情书，可以体验操控多个从者，多样的技能、爽快的无双体验和出色的剧情，王朝了型月。

# **S4U: 都市朋克2011与爱的重拳 9/10**

出色的类酒保行动视觉小说，核心玩法非常有趣，剧情也很不错。

# **全网公敌2 新世界 8/10**

维持着前作的高水准，今年更新的结局有点缺憾且又在挖坑。

# **ENDER MAGNOLIA: Bloom in the Mist 6/10**

太出色的美术和音乐，核心玩法也没什么问题，但是数值问题过分严重，极度影响体验。

# **巴别塔圣歌 8/10**

出色的机密玩法，要长脑子了，但是难度曲线不是很合理。

# **Untitled Goose Game 5/10**

制作组欺负我，我欺负NPC，但我是好鹅。

# **Sid Meier’s Civilization® VI 10/10**

无需多言的好玩，停不下来的好玩。

# **双人成行 9/10**

终于把这个玩通关了，出色的关卡和能力设计，合作玩法恰到好处，就是流程有点短。

# **迷失之径 2/10**

重复且少量的构筑方向，同质化的内容，平衡不好的藏品解锁，烂。

# **中国式相亲 2/10**

和上面那个坐一桌，打牌公式，角色魅力塑造垃圾。

# **Streets of Rogue 5/10**

够混乱，但是不够好玩；内容多，但是游玩路径模糊。

# **Portal 9/10**

曾经的天才之作。

# **猫神牧场 6/10**

公式作品获取及格分。

# **双影奇境 10/10**

更进一步的关卡设计，更强的合作，最终的创意爆发更是顶级。

# **AI LIMIT 无限机兵 10/10**

年度出色地图设计，虽然战斗设计略显青涩，但是很好去魂糟粕，取其精华。

# **超级键盘侠 7/10**

你要做的就是把大脑扔掉然后享受迷因快感。

# **深空梦里人 8/10**

出色视觉风格，有趣的角色扮演体验，还算不错的剧情，只是后期各方面都有点令人失望。

# **优米雅的炼金工房 ～追忆之炼金术士与幻创之地～ 3/10**

你把钱退给我吧，我再也不预购了，育碧都不敢这么做游戏的。

# **闰跃之年 Leap Year 10/10**

无敌的知识锁设计，巧妙的核心机制利用与扩展，纯粹的教科书。

# **夏末盛开的花 9/10**

出色的剧情、人设，不错的美术与音乐，最重要的是它只要六块钱。

# **九日 3/10**

想做的越多，就做的越不好。

# **界外狂潮 1/10**

一手好牌打的稀烂。

# **What Remains of Edith Finch 8/10**

不赖的步行模拟与剧情，演出挺不错的。

# **光与影：33号远征队 1/10**

年度最失望、最过誉、最冲击波、最垃圾剧情、编剧最该去死、玩法策划也该去死游戏。

# **救命！怎么又塌了! 4/10**

核心玩法真的不错，但是肉鸽体验太烂了，内容重复度太高了。

# **Banana Shooter 7/10**

其实是Call of Banana。

# **激射超新星 5/10**

固定的类吸血鬼幸存者，但是做不出有趣的构筑。

# **PUBG: BATTLEGROUNDS 2/10**

枪械手感真是依托，节奏又慢又慢又快，玩着一股异味。

# **万手一体 9/10**

打牌和叙事的有机结合，出色的视觉风格与配乐，好玩就能概括它的全部。

# **Resident Evil 4 10/10**

太无敌的箱庭设计，探索与线性平衡的恰到好处，枪械多样，恐怖氛围到位。

# **星之侵略者 7/10**

杀戮尖塔与陷阵之志的巧妙融合，但是游戏后期有点同质化。

# **DOOM 8/10**

血脉贲张，但是又不够那么爽。

# **多重人生 10/10**

基建、探索、生存、道德选项、剧情的有机融合，出色的配乐与视觉风格让它更上一层楼。

# **泡姆泡姆 5/10**

一般的合作玩法，公式谜题设计，后期真的枯燥。

# **卡牌少女不会受伤 Trizon 2/10**

你要知道不是有美少女大家就会买账的。

# **PETS 9/10**

也是出色的知识锁设计，只是部分谜题我的脑子真的有点不够。

# **剑星 5/10**

你要知道不是有美少女大家就会买账的，别的地方太公式且做的不好玩了，最后反派出现的演出也唐玩了，能不能好好设计一下韩国男角色。

# **Ravenswatch 鸦卫奇旅 4/10**

玩起来真的很累，作为肉鸽局内养成又不好玩深度不够，数值问题非常严重，只看到了我和朋友痛苦的脸。

# **PEAK 10/10**

伟大的体力条设计，足够的合作体现，乐趣十足的关卡，更新也到位，或将永远存在于我的多人游戏库里。

# **魔法少女的魔女审判 8/10**

还算可以和解的剧情，前期案子有点太降智，人设都很不赖。

# **孤星猎人 4/10**

同质化还是同质化！底子那么好但是后期一点让人不想玩。

# **Slay the Princess — The Pristine Cut 9/10**

很独特的视觉小说，多结局与抽丝剥茧到最终结局的过程让人欲罢不能，手绘画风太给力。

# **Patrick's Parabox 9/10**

递归和逆向，但我无法成为推箱子之王。

# **矩阵：零日危机 1/10**

你回去做你的生死狙击去吧。

# **黑神话：悟空 7/10**

本质Boss Rush，地图设计真有点连累了，最后一章更是拉低评分，动作设计和美术倒是挺在线的。

# **枪豆人 8/10**

核心玩法有趣，画风可爱，联机节目效果十足。

# **虚空破坏者 6/10**

Bro开创了类魂射击肉鸽（并不是），但是体验就像前面的这几个词。

# **我在地府打麻将 - Demonic Mahjong 6/10**

爽还是爽的，但是角色真的有点丑了，以及后期玩法同质化和数值膨胀严重。

# **Hollow Knight: Silksong 9/10**

痛并快乐着，等待是值得的。

# **真・三国无双 起源 6/10**

优化真的不是很好，不过核心体验和角色扮演真的不错，但是无双部分都不如FSR。

# **呓语小镇 8/10**

我有强迫症！

# **Ghost of Tsushima导演剪辑版 7/10**

美味的罐头，战斗玩法简单有趣，出色的视觉风格，但是还是罐头。

# **战地风云™ 6 9/10**

所有人都要玩点好的。

# **BALL x PIT — 球比伦战记 9/10**

今年唯一一个挑不出毛病的肉鸽游戏，只是前期开荒有点折磨。

# **四叶草深渊 CLOVERPIT 4/10**

噱头大于游戏内容，数值碰撞与藏品污染问题严重。

# **逃离鸭科夫 1/10**

本质纯清单游戏，作为单机游戏甚至学来了三角洲的问题，神了。

# **GOODBYE WORLD  9/10**

做游戏还是好玩的？做游戏还是好玩的！

# **Keep Driving 6/10**

本质卡组构建和资源管理游戏，公路旅行噱头大于体验，但是音乐真的好听！

# **FINAL FANTASY VII REBIRTH 3/10**

我和JRPG和解不了。

# **Lies of P 8/10**

出色的视觉风格，优秀的地图设计，但是差就差在恶意设计的强制保留。

# **Dispatch 9/10**

你知道的，我一直都喜欢假小子的，当然我也很喜欢这种调兵遣将的玩法。

# **Marvel’s Spider-Man: Miles Morales 5/10**

短，贵，荡蛛丝好玩。

# **梦之形 5/10**

雨中冒险没有那么好模仿。

# **海默2000 10/10**

检索玩法很独特，随着检索推动并拼凑的剧情也很动人。

# **编程农场 9/10**

除了强制锁天赋树没啥毛病。

# **这里没有兽人 6/10**

一个适合你点进去玩个一把就掌握所有的游戏，但也局限于此。

# **奇唤士 8/10**

我是华沙雨宫莲！

# **神经鹅 6/10**

精神污染，但是游戏内容设计左脑反驳右脑，前中期还是好玩的。

# **Resident Evil 2 8/10**

略显青涩的关卡设计，恐怖氛围倒是很到位。

# **死亡日：狂杀末路 2/10**

至今仍然想不明白为什么类吸血鬼幸存者游戏要刁难玩家的爽感。

# **空之轨迹 the 1st 8/10**

我又和JRPG和解了！我要玩scd。

# **Apex Legends™ 8/10**

更快、更好、更强以及更少的挂

# **Valorant 6/10**

好玩，但是设计师当英雄联盟做它，开挂炸鱼的又太多。

# **Counter-Strike 2 7/10**

经典永不过时，但是依旧一堆开挂炸鱼的。

# **三角洲行动 0/10**

吃大粪去吧。

# **HELLDIVERS™ 2 10/10**

保卫超级地球我是自愿的。

# **Mgn年度最佳独立游戏**

空洞骑士：丝之歌

# **Mgn年度最佳艺术风格**

多重人生

# **Mgn年度最佳关卡设计**

Resident Evil 4

# **Mgn年度最佳游戏剧本**

空之轨迹 the 1st

# **Mgn年度最佳多人游戏**

PEAK

# **Mgn年度最佳角色扮演**

Fate/Samurai Remnant

# **Mgn年度最佳游戏机制**

PEAK体力条

# **Mgn年度最佳游戏配乐**

光与影：33号远征队

# **Mgn年度游戏**

多重人生

# **Mgn年度最令人失望游戏**

光与影：33号远征队
